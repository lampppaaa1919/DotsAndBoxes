# Проект по Вештачка Интелигенција - Dots And Boxes (Точки и кутии)

Овој проект е имплементација на игра којашто се одвива во повеќе-агентни простори применувајќи генетски алгоритам

Играта се одвива така што секој играч црта една линија помеѓу две соседни точки. Доколку играчот го нацрта последниот ѕид на некоја кутија, ја освојува кутијата и добива еден поен. Играчот со најмногу поени победува.

Алгоритмот ќе научи како играчот да го победи противникот.
